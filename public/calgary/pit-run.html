Pit Run in Calgary — Bank-Run Gravel for Fill, Base, and Driveways

Pit run is the most cost-efficient bulk aggregate you can buy in Calgary — unscreened gravel straight from the pit, with the natural mix of stone sizes and fines that nature put there. It's the default for structural fill, driveway base, and any application where you don't need a tightly graded product.

What pit run is

Pit run is gravel pulled directly from a pit and delivered with no screening. The mix of stone sizes (from fines through cobbles) is whatever the pit's natural deposit contains. This makes it cheaper than crushed or screened products but less consistent.

Common Calgary uses

  • Structural fill — load-bearing fill under foundations and slabs (with engineering review).
  • Driveway base — sub-layer under road crush.
  • Acreage and rural roads — economical surface for low-traffic driveways.
  • Trench backfill — where compactable, drainable fill is needed.

When to use pit run vs. road crush

  • Pit run: thick base layers, structural fill, rural driveway surfaces.
  • Road crush: finished driveway and parking surfaces, paver bases, shed pads.

Many Calgary driveways use both — pit run as a deep base, road crush as the finish layer.

Frequently asked questions

What's the difference between pit run and road crush?
Pit run is unscreened and contains a natural size mix straight from the pit. Road crush is screened and crushed to a 20mm-minus spec. Pit run is cheaper; road crush is more consistent.
Is pit run good for driveways?
Yes for rural acreage and base layers. For finished urban driveways, road crush over pit run is the typical spec.
How much does pit run cost in Calgary?
Pricing varies by pit. Live listings show current pricing.
Can I use pit run for a shed pad?
Yes as a base layer. Top with 4" of road crush for a finished surface.
Does pit run compact well?
Yes — the natural fines and varied sizes interlock when compacted.
